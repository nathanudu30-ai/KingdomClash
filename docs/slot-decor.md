# Décors du slot sans fichiers binaires

Le projet n'embarque plus d'images binaires pour le décor du slot.

Configurez simplement vos URLs d'images dans `src/config/slotDecor.ts`:
- `backgrounds` (plusieurs fonds possibles)
- `machineFrame`
- `spinButton`

Vous pouvez héberger vos images sur Supabase Storage, Cloudinary, un CDN ou GitHub raw.
